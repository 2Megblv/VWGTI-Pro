The following patterns guide extraction without logic changes:

**VolumeProfile.mqh:** Encapsulates 400-bin distribution calculation, POC/VAH/VAL identification, HVN/LVN detection, multi-level proration logic.

**Main EA:** Orchestrates via #include directives; OnTick calls extracted functions without reimplementing calculation logic.
